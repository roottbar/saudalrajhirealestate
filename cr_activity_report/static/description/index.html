Features

  • Goal is to generate Excel and PDF report of Activity of User based on Due Date, Type of Activity and Range.
  • Easily generate the Excel or PDF report of activity of user.
  • An ability to easily get follow-up of user's Activity in PDF or Excel Formate.

Generate Excel & PDF Report

  • Click on menu named Sales under Main menu.
  • Under Action menu user will see menu named as Activity Report.


  • On clicking on it user will see a wizard containing 4 Fields asking Activity Type, Due Date, Range and User Id followed with 3 button name Excel Report, PDF Report and Cancel.

  • When user fills all asked details and on clicking Excel Report or PDF Report button user will have a Excel Report or PDF Report Downloaded with data.
