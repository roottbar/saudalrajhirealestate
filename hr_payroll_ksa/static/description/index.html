Overview

Payroll application working on management of employee regular salary in the company, based on the popular saudi contracts as the basics, allowances & deductions.

This module needs small configuration, like implement the rules calculation like housing allowance per working days to be look like:
result = (contract.hra / 30) * worked_days.WORK100.number_of_days

or calculate a percentage from of the basic salary (contract.wage)
result = ((contract.wage * 0.20) / 30) * worked_days.WORK100.number_of_days

Features

  • Define your salary structures.
  • Config salary rules in simple steps.
  • Insert National Address Details with uploading verification certificate.
  • Generate Payslip batches in easy view.
  • Create & generate Employee Payslips in Seconds.
